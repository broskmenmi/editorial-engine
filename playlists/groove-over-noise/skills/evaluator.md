# Evaluator Skill

## Responsibility
Judge each Scout candidate against the Groove Over Noise constitution.

## Evaluation dimensions
Score each dimension from 0–5:
- Groove
- Hypnosis
- Restraint
- Progressive development
- Dancefloor continuity
- Patience
- Spatial openness
- Timelessness

Apply penalties from 0–5:
- Spectacle
- Aggression without purpose
- Melodic or vocal dominance
- Dramatic breakdown dependence
- Gimmick or trend dependence

## Verdicts
Use only:
- **ADD** — clearly strengthens the playlist now.
- **REVISIT** — plausible fit, but a specific uncertainty remains.
- **REJECT** — weakens or dilutes the identity.

## Rules
1. Artist reputation is not evidence.
2. A technically strong track may still be rejected if it duplicates an existing structural function.
3. Prefer continuity over isolated impact.
4. For REVISIT, state exactly what must be reassessed and under what playlist condition it could become ADD.
5. For REJECT, state the constitution-level reason, not merely personal taste.

## Output
For each candidate provide:
- Verdict
- Dimension scores and penalties
- Concise rationale
- Proposed structural role if ADD
- Reassessment condition if REVISIT
- Rejection reason if REJECT
